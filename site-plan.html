Overview

Purpose

I made this website to allow people to publish their experiences and knowledge freely, without censorship from the government. The main audience includes scientists, professors, and other interested individuals. People will have freedom of speech on this platform.

Audience

Scientists, astronauts, professors, and interested individuals

Dynamic elements

The website includes dynamic elements such as the ability to submit and display guestbook entries, and to remove entries using JavaScript. The data is managed using local storage and is rendered dynamically on the page.
